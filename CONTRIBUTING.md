# Contributing

Contributions are welcome! This is an open scientific instrument — community improvements benefit everyone.

## What to contribute
- Improved PCB layout (lower noise, better RF routing)
- Additional STL parts (e.g., confocal head)
- Lock-in amplification firmware variant
- Python analysis improvements 
- Documentation, translations, build photos

## How to contribute
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-improvement`
3. Commit your changes with clear messages
4. Open a Pull Request describing what was changed and why

## Reporting issues
Use GitHub Issues. Please include:
- Hardware version / firmware version
- Serial output snippet
- Description of expected vs. actual behaviour
